# PynqC診断

入力由来の問題はpanicせず`Diagnostic`として返し、CLIを非0で終了する。

```text
examples/c/error.pc:4:9: エラー[名前解決]: 未定義変数 `value` です

    value = 42;
    ^^^^^

補足: `value`を使用前に宣言してください
```

診断にはファイル名、1始まりの行・列、分類、ソース行、Span幅のキャレット、任意の補足を含む。
分類は字句解析、構文解析、名前解決、型、コード生成、外部ツールである。

代表例は未終了文字列・コメント、不正文字、型名や区切り不足、未定義名、同一スコープ重複、
関数引数不一致、void値使用、不正return、非左辺値代入、非ポインタ間接参照、不正ポインタ演算、
ループ外break/continueである。`examples/c/negative`と`test_c_negative.sh`は、失敗するだけでなく
各診断に期待する語が含まれることを検査する。
